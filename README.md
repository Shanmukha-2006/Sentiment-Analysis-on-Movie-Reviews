# 🎬 Sentiment Analysis on Movie Reviews

## About the Project

This project performs **Sentiment Analysis** on movie reviews using **Natural Language Processing (NLP)** and **Machine Learning** in **Google Colab**. It classifies movie reviews as **Positive** or **Negative** by analyzing the text and extracting meaningful features.

## Features

* Data cleaning and preprocessing
* Text preprocessing using NLP
* Sentiment classification
* Exploratory Data Analysis (EDA)
* Data visualization
* Model evaluation

## Technologies Used

* Google Colab
* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* Matplotlib
* Seaborn

## Dataset

The project uses the **IMDb Movie Reviews Dataset**, which contains movie reviews labeled as **Positive** and **Negative**.

## Project Workflow

1. Import the dataset into Google Colab
2. Clean and preprocess the text
3. Convert text into numerical features
4. Train the machine learning model
5. Evaluate the model
6. Predict the sentiment of movie reviews

## Results

The analysis provides insights into:

* Distribution of positive and negative reviews
* Common words in movie reviews
* Model performance on sentiment classification
* Overall sentiment trends

## Future Improvements

* Use Deep Learning models (LSTM/BERT)
* Build a Streamlit web application
* Deploy the model online
